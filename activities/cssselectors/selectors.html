ENGAGEMENT QUESTIONS:

List out the different ways to select an item.

  1. Tag Name
  2. ID - #
  3. Class - .
  4. Descendant selector

What is the difference between an ID and a Class?

An ID should be unique, you can't reuse them. A Class can be reused. *A tag can have one or more class.

What is the difference between a SPAN tag and a DIV tag?

The SPAN tag is an inline element while the DIV tag is a block-level element.

What is the difference between a DIV tag and HEADER, MAIN, FOOTER?

A DIV tag is a ganeric container with no semantic meaning. The HEADER, MAIN, FOOTER are semantic elements that describe their purpose in the document structure.

What are different ways to select a color?

  1. Color keywords: red, green, blue
  2. Hexadecimal: #FF0000
  3. RGB: rgb(255, 0, 0)
