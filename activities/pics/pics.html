Engagement Questions:

1. What is the difference between a block and an inline tag? Give examples of each one.

Block elements start on a new line and take up the full width available. Ex: div, p, headings, unordered and ordered lists.

Inline elements do not start on a new line and only take up as much width as necessary. They flow "in line" with the surrounding content. Ex: span, anchor links, bold and italics, image.

2. What is normal flow in HTML?

Normal flow refers to the default layout behavior of HTML elements before any CSS positioning is applied.

3. What does float do? Why do you need to clear the float?

The float CSS property moves an element to the left or right of its container, allowing text and inline elements to wrap around it.

4. What do absolute, fixed, and relative do?

"Absolute" removes the element from normal flow and positions it relative to its nearest positioned ancestor. "Fixed" removes the element from normal flow and positions it relative to the viewport. "Relative" keeps the element in normal flow but allows offsetting from its normal position.
