Engagement Questions:

1. Why don't we use the border property?

Nowadays, CSS is more widely used to control how your content looks, including borders. Using CSS for borders is more flexible, modern, and aligns with best practices in web development.

2. What is the easiest way to color every other row a different color?

The easiest way is by using the 'nth-child()' tag.

3. When should we use a table?

You use a table when you have a data to display.
