Internet

The Internet refers to a network of interconnected computers all around the globe. This allows them to share information with one another.

Web

The Web is a part of the Internet where websites and web pages can be found. It is a system of linked multimedia contents and hypertext documents that can be accessed through web browsers in the Internet.

How are the Internet and Web different?

The internet pertains to the whole global network of interconnected computers, while the Web merely refers to the service that works over the Internet that gives its users the ability to access and share media through websites.

ASCII and Unicode

ASCII stands for the American Standard Code for Information Interchange. It is a standard for encoding characters that uses numeric codes to represent mainly English characters.

On the other hand, Unicode is another standard for character encoding that could represent a broader set of characters from various languages and scripts, making it more versatile than ASCII.

What is UTF-8?

UTF-8, also known as the 8-bit Unicode Transformation Format, is a way to encode text that can handle every character in the Unicode character set.

URL

The Uniform Resource Locator (URL) refers to the address used to acces resources on the Web. It is the one you type into the web browser to access a specific page or file.

Domain Name

The easy-to-remember name for a website's address is called the Domain Name. It usuall ends with a '.com' or '.org'

HTTP

The Hypertext Transfer Protocol (HTTP) is a set of rules used for transferring web pages on the Internet.

HTTPS

The Hyptertext Transfer Protocol Secure (HTTPS) is a secure version of HTTP that encrypts the data being transferred across the Web.

IP Address

IP Address or Internet Protocol Address is a unique numeric address that is used to identify each device or computer connected to the Internet.

DNS

The Domain Name System (DNS) is a system that matches domain names to IP Addresses.

W3C Recommendations

These are official rules and standards set by the Word Wide Web Consortium (W3C) to ensure that the Web works uniformly across different systems.

XML, XHTML, HTML, HTML5

The eXtensible Markup Language (XML) uses the tags that you define as a way to format and organize data.

The eXtensible Hyptertext Markup Language (XHTML) is a stricter version of HTML that follows the rules of XML to ensure better structure and compatability.

Hypertext Markup Language (HTML) is the standard language used to create and design web pages. Its latest version is known as HTML5 which introduces new tools and features for better web development and multimedia support.
