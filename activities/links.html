Engagement Questions:

Explain the difference between an absolute link and a relative link. Why would you use relative links?

    An absolute link contains the complete URL of the website or page. A relative link is in your website only. You use it to link different parts of your website.

List the 3 rules with relative links:
  • . means the folder that I'm in / current folder
  • .. means the parent folder (go up a folder)
  • / start at the root
Do links always need to be inside a NAV tag?
