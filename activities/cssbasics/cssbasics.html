Engagement:

1. Explain the difference between inline, embedded, and external CSS.

Inline CSS is applied directly to individual HTML elements using the style attribute. DO NOT USE THIS IN CLASS.

Embedded CSS involves placing style rules within the style tag in the head section of an HTML document. STILL, DO NOT USE THIS IN CLASS.

External CSS is involves creating a separate .css file and linking it to HTML documents using the link tag.

2. Explain why you should only use external CSS.

Most websites are generated by a program. To make it easier to edit webpages, use a separate file which is the external css.

3. What is a selector used for?

Selectors are the bridge between your HTML structure and CSS styles, allowing you to precisely control the appearance of web page elements.
