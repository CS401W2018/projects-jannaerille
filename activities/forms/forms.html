Engagement Questions:

1. What is JavaScript? How did it get its name?

JavaScript is a scripting language that allows developers to implement complex features on webpages. It's the third layer of standard web technologies, alongside HTML and CSS.

JavaScript was originally called "Mocha", then "LiveScript" when it was developed by Netscape in 1995. It was renamed to JavaScript, likely as a marketing tactic to capitalize on the popularity of Java at the time. Despite the similar names, JavaScript is not related to Java.

2. What are the two ways you can run JavaScript?

JavaScript can be run by client-side in web browsers or server-side using Node.js or similar runtime environments.

3. What was the most popular front-end js library?

jQuery was the most popular library.

4. What is the most popular front-end js library?

Now, React is the most popular.
